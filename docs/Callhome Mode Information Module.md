Synopsis
--------
This module will retrieve VxRail CallHome Mode information, whether it is muted or unmuted.

Supported Endpoints
--------

* GET /callhome/mode

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_callhome_getmode.py calls any existing version of GET /callhome/mode API
- Details on execution of module dellemc_vxrail_callhome_getmode.py can be checked in the logs /tmp/vxrail_ansible_callhome_getmode.log


Examples
--------

``` yaml+jinja
 - name: Retrieves VxRail Callhome Mode Information, version specified by api_version_number
    dellemc_vxrail_callhome_getmode:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        timeout : "{{ timeout }}"
        api_version_number: "{{ api_version_number }}"
```
