Synopsis
--------
This module will update the segment configuration for a specific segment.

Supported Endpoints
--------
* POST /cluster/layer3/segment/{segment-label}

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_cluster_layer3_update_segment.py can call any existing version of /cluster/layer3/segment/{segment_label} API
- Details on execution of module dellemc_vxrail_cluster_layer3_update_segment.py can be checked in the logs /tmp/vxrail_ansible_layer3_update_segment.log
