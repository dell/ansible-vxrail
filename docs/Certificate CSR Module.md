Synopsis
--------
This module will generate a Certificate Signing Request with the given information.

Supported Endpoints
--------
* POST /certificates/csr

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_certificate_generate_csr.py can call any existing version of /certificates/csr API  
- Details on execution of module dellemc_vxrail_certificate_generate_csr.py can be checked in the logs /tmp/vxrail_ansible_certificate_generate_csr.log

Examples
--------

``` yaml+jinja
  - name: Generate a Certificate Signing Request
    dellemc_vxrail_certificate_generate_csr:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        country: "{{ country }}"
        state: "{{ state }}"
        locality: "{{ locality }}"
        organization: "{{ organization }}"
        organization_unit: "{{ organization_unit }}"
        common_name: "{{ common_name }}"
        email_address: "{{ email_address }}"
        subject_alt_name: "{{ subject_alt_name }}"
        api_version_number: "{{ api_version_number }}"
```
