Synopsis
--------

This module will validate a L2 node expansion, perform a L2 node expansion
based on the provided expansion specification and query the status.

Supported Endpoints
--------

* POST /cluster/expansion/validate
* POST /cluster/expansion
* GET /requests/{id}

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_cluster_expansion.py calls any existing version of Post /cluster/expansion API
- Details on execution of module dellemc_vxrail_cluster_expansion.py can be checked in the logs /tmp/vxrail_ansible_cluster_expansion.log


Examples
--------

``` yaml+jinja
 - name: Start a cluster expansion, version specified by api_version_number
    dellemc_vxrail_cluster_expansion:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        vxm_version: "{{ vxm_version }}"
        host_psnt: "{{ host_psnt }}"
        hostname: "{{ hostname }}"
        mgt_account: "{{ mgt_account }}"
        mgt_passwd: "{{ mgt_passwd }}"
        root_passwd: "{{ root_passwd }}"
        mgt_ip: "{{ mgt_ip }}"
        vsan_ip: "{{ vsan_ip }}"
        vmotion_ip: "{{ vmotion_ip }}"
        mgt_ipv6: "{{ mgt_ipv6 }}"
        vsan_ipv6: "{{ vsan_ipv6 }}"
        vmotion_ipv6: "{{ vmotion_ipv6 }}"
        rack_name: "{{ rack_name }}"
        order_number: "{{ order_number }}"
        maintenance_mode: "{{ maintenance_mode }}"
        timeout: "{{ timeout }}"
        api_version_number: "{{ api_version_number }}"
```
