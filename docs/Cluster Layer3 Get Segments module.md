Synopsis
--------
This module will get a list by cluster layer3 segments.

Supported Endpoints
--------
* GET /cluster/layer3/segments

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_cluster_layer3_get_segments.py can call any existing version of /cluster/layer3/segments API
- Details on execution of module dellemc_vxrail_cluster_layer3_get_segments.py can be checked in the logs /tmp/vxrail_ansible_layer3_get_segments.log

Examples
--------

``` yaml+jinja
    - name: Get Cluster Layer3 Segments
      dellemc_vxrail_cluster_layer3_get_segments:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        api_version_number: "{{ api_version_number }}"
```
