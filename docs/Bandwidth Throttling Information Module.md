Synopsis
--------
This module will retrieve bandwidth throttling information.

Supported Endpoints
--------
* GET /system/bandwidth-throttling

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_bandwidth_throttling_info.py can call any existing version of /system/bandwidth-throttling API
- Details on execution of module dellemc_vxrail_bandwidth_throttling_info.py can check Log file /tmp/vxrail_ansible_bandwidth_throttling_info.log

Examples
--------

``` yaml+jinja
  - name: Retrieve bandwidth throttling Information.
    dellemc_vxrail_bandwidth_throttling_info:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        timeout : "{{ timeout }}"
        
  - name: Retrieve bandwidth throttling Information. Version specified by api_version_number.
    dellemc_vxrail_bandwidth_throttling_info:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        timeout : "{{ timeout }}"
        api_version_number "{{ api_version_number }}"

```
