Synopsis
--------

This module will download VxRail Support Logs by Id.

Supported Endpoints
-------------------

* GET support/logs/{log_id}/download

Notes
-----

- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_support_logs_id_download.py calls any existing version of Get support/logs/{log_id}/download API
- Details on execution of module dellemc_vxrail_support_logs_id_download.py can be checked in the logs /tmp/vxrail_log_support_logs_id_download.log

Examples
--------

```yaml+jinja
- name: Download VxRail Support Logs, version specified by api_version_number
   dellemc.vxrail.dellemc_vxrail_support_logs_id_download:
     vxmip: "{{ vxmip }}"
     vcadmin: "{{ vcadmin }}"
     vcpasswd: "{{ vcpasswd }}"
     log_id: "{{ log_id }}"
     output_file_path: "{{ output_file_path }}"
     timeout : "{{ timeout }}"
     api_version_number: "{{ api_version_number }}"
```
