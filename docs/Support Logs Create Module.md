Synopsis
--------

This module will create a VxRail Support Log.

Supported Endpoints
-------------------

* POST /support/logs

Notes
-----

- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_support_logs.py calls any existing version of Post /support/logs API
- Details on execution of module dellemc_vxrail_support_logs.py can be checked in the logs /tmp/vxrail_support_logs.log

Examples
--------

```yaml+jinja
 - name: Create a VxRail Support Logs, version specified by api_version_number
   dellemc.vxrail.dellemc_vxrail_support_logs:
     vxmip: "{{ vxmip }}"
     vcadmin: "{{ vcadmin }}"
     vcpasswd: "{{ vcpasswd }}"
     types: "{{ types }}"
     nodes: "{{ nodes }}"
     autoclean: "{{ autoclean }}"
     timeout : "{{ timeout }}"
     api_version_number: "{{ api_version_number }}"
```
