Synopsis
--------
This module will change the VxRail Manager IP address.

Supported Endpoints
--------
* Post /system/vxm-ipaddress

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_system_vxmipaddress_change.py calls any existing version of Post /system/vxm-ipaddress API
- Details on execution of module ellemc_vxrail_system_vxmipaddress_change.py can be checked in the logs /tmp/vxrail_ansible_system_vxmipaddress_change.log


Examples
--------

``` yaml+jinja
  - name: Change the VxRail vCenter server IP address
    dellemc_vxrail_change_vcipaddress:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        vcroot: "{{ vcroot }}"
        vcroot_passwd: "{{ vcroot_passwd }}"
        ip: "{{ ip }}"
        gateway: "{{ gateway }}"
        netmask: "{{ netmask }}"
        ipv6: "{{ ipv6 }}"
        gateway_ipv6: "{{ gateway_ipv6 }}"
        prefix_length_ipv6: {{ prefix_length_ipv6 }}
```
