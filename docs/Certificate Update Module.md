Synopsis
--------
This module will update the VxRail Manager certificate.

Supported Endpoints
--------
* POST /certificates/import-vxm

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_certificate_update.py can call any existing version of /certificates/import-vxm API
- Details on execution of module dellemc_vxrail_certificate_update.py can be checked in the logs /tmp/vxrail_ansible_certificate_update.log

Examples
--------
``` yaml+jinja
  - name: Update the VxRail Manager certificate with v2/certificates/import-vxm API
    dellemc_vxrail_certificate_update:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        cert: "{{ cert }}"
        root_cert_chain: "{{ root_cert }},{{ intermediate_cert }}"
        private_key: "{{ private_key }}"
        password: "{{ password }}"
        timeout: "{{ timeout }}"
        api_version_number: 2
```
``` yaml+jinja
  - name: Update the VxRail Manager certificate with v3/certificates/import-vxm API
    dellemc_vxrail_certificate_update:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        cert: "{{ cert }}"
        root_cert_chain:
         - "{{ root_cert }}"
         - "{{ intermediate_cert }}"
        private_key: "{{ private_key }}"
        password: "{{ password }}"
        vc_admin_account: "{{ vc_admin_account }}"
        vc_admin_password: "{{ vc_admin_password }}"
        timeout: "{{ timeout }}"
        api_version_number: 3
```
