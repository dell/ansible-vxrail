Synopsis
--------
This module will change the VxRail cluster EVC mode.

Supported Endpoints
--------
* POST /cluster/evc

Notes
-----
- Make sure your VxRail environment supports the API that you use
- Module dellemc_vxrail_cluster_evc_change.py can call any existing version of /cluster/evc API
- Details on execution of module dellemc_vxrail_cluster_evc_change.py can check Log file /tmp/dellemc_vxrail_cluster_evc_change.log

Examples
--------

``` yaml+jinja        
  - name: Changes cluster EVC mode. Version specified by api_version_number.
    dellemc_vxrail_cluster_evc_change:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        evc_mode: "{{ evc_mode }}"
        timeout : "{{ timeout }}"
        api_version_number: "{{ api_version_number }}"
```
