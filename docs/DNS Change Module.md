Synopsis
--------
This module will change the DNS servers for the cluster.

Supported Endpoints
--------

* POST /system/dns

Notes
-----
- This module calls any existing version of the /system/dns API, please ensure your VxRail cluster supports this API.
- Can check Log file /tmp/vxrail_ansible_system_dns_change.log for more details about execution result.


Examples
--------

``` yaml+jinja
  - name: Changes the VxRail DNS settings. Version specified by api_version_number
    dellemc_vxrail_system_dns_change:
        vxmip: "{{ vxmip }}"
        vcadmin: "{{ vcadmin }}"
        vcpasswd: "{{ vcpasswd }}"
        component: "{{ components }}"
        servers: "{{ servers }}"
        api_version_number "{{ api_version_number }}"
        timeout: "{{ timeout }}"
```
